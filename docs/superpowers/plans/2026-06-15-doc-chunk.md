# doc_chunk 文档提取与智能分块 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建可安装的 `doc_chunk` Python 包（CLI `doc-chunk`），实现 Word/PDF 五阶段流水线：extract → outline → outline-refine → chunk → enrich。

**Architecture:** 单包按流水线阶段划分子模块；Pydantic 模型即 JSON schema；`api.py` 为 skills 稳定入口，CLI 薄封装；从 tender_doctor/tender_knowledge 移植算法但不添加 pip 依赖；LLM 通过 `LLMClient` 协议注入。

**Tech Stack:** Python 3.11+, python-docx, lxml, pymupdf, pydantic v2, typer, pyyaml, pytest, jsonschema, openai (compatible)

**需求来源:** [`docs/superpowers/specs/2026-06-15-doc-chunk-requirements.md`](../specs/2026-06-15-doc-chunk-requirements.md)

```python
from __future__ import annotations
from typing import Any, Literal
from pydantic import BaseModel, Field

class SourceInfo(BaseModel):
    path: str
    file_name: str
    file_type: Literal["docx", "doc", "docm", "pdf"]
    title: str | None = None

class StageStatus(BaseModel):
    status: Literal["success", "partial_success", "failed", "skipped"] = "success"
    started_at: str | None = None
    finished_at: str | None = None
    warnings: list[str] = Field(default_factory=list)

class Manifest(BaseModel):
    schema_version: Literal["1.0"] = "1.0"
    status: Literal["success", "partial_success", "failed"] = "success"
    source: SourceInfo
    stages: dict[str, StageStatus] = Field(default_factory=dict)
    outputs: dict[str, str] = Field(default_factory=dict)
    warnings: list[str] = Field(default_factory=list)
    errors: list[str] = Field(default_factory=list)
```

```python
# src/doc_chunk/workspace/layout.py
from __future__ import annotations
from dataclasses import dataclass
from pathlib import Path
import shutil
from doc_chunk.errors import WorkspaceError

@dataclass(frozen=True, slots=True)
class OutputWorkspace:
    root: Path
    content_path: Path
    manifest_path: Path
    images_dir: Path
    chunks_dir: Path
    logs_dir: Path

    @classmethod
    def create(cls, root: Path, *, overwrite: bool) -> OutputWorkspace:
        if root.exists() and not overwrite:
            raise WorkspaceError(f"Output workspace already exists: {root}")
        if root.exists() and overwrite:
            shutil.rmtree(root) if root.is_dir() else root.unlink()
        images_dir = root / "images"
        chunks_dir = root / "chunks"
        logs_dir = root / "logs"
        root.mkdir(parents=True, exist_ok=True)
        images_dir.mkdir(exist_ok=True)
        chunks_dir.mkdir(exist_ok=True)
        logs_dir.mkdir(exist_ok=True)
        return cls(
            root=root,
            content_path=root / "content.md",
            manifest_path=root / "manifest.json",
            images_dir=images_dir,
            chunks_dir=chunks_dir,
            logs_dir=logs_dir,
        )
```

`manifest_io.py`: `load_manifest(path) -> Manifest`, `save_manifest(ws, manifest)`.

```python
# src/doc_chunk/extract/detect.py
from pathlib import Path
from doc_chunk.errors import UnsupportedFormatError

_SUFFIX_MAP = {".docx": "docx", ".doc": "doc", ".docm": "docm", ".pdf": "pdf"}

def detect_file_type(path: Path) -> str:
    suffix = path.suffix.lower()
    if suffix not in _SUFFIX_MAP:
        raise UnsupportedFormatError(f"Unsupported file type: {suffix}")
    return _SUFFIX_MAP[suffix]
```
